Charity Web

CharityWeb is a proposed solution to the various problems concerning online donations in the world today. In a nutshell, it is a web application that will enable its users to interact directly with various charity organizations. In other words, it will allow the users to not only donate but give them a proper description of what these charities stand for and show evidence of their works. It will also link them directly to the websites of these organizations thereby reducing the number of fake charities with fraudulent causes.
